# Cordas e Música
Projeto criado para uma atividade do curso de Desenvolvimento de Sistemas do SENAC. A atividade em questão é o Projeto Integrador presente na UC15.
O sistema “Cordas e Música” é pensado para uma loja de guitarras gerida por apenas uma pessoa, ele permitirá o registro e gerenciamento de produtos,
clientes e vendas, proporcionando controle eficiente e organizado das transações comerciais.

## Status
Concluído

## Tecnologias Utilizadas

- MySQL

## Funcionalidades
- Cadastro de Produto

Requisito responsável por cadastrar os produtos após preenchimento de todos os campos: nome, marca, modelo, ano de fabricação, cor e preço.
- Cadastro de clientes

Requisito responsável por cadastrar os clientes após preenchimento de todos os campos: nome, CPF, telefone e Email.
- Registro de Venda

Requisito responsável por registrar os produtos vendidos bem como o cliente que comprou e a data da venda.
