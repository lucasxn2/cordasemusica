# Cordas e Música
Projeto criado para uma atividade do curso de Desenvolvimento de Sistemas do SENAC. A atividade em questão é o Projeto Integrador presente na UC15.
O sistema “Cordas e Música” é pensado para uma loja de guitarras gerida por apenas uma pessoa, ele permitirá o registro e gerenciamento de produtos,
clientes e vendas, proporcionando controle eficiente e organizado das transações comerciais.

## Tecnologias Utilizadas
- Java
- MySQL
